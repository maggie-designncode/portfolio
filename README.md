## Case Studies

### Case Study 1 — Digitizing High-Friction Bond Trading
**JPMorgan Execute · Zero-to-one · Client-facing**

Building the first electronic trading tool for BWIC (Bid Wanted in Competition) bond trading on JPMorgan's single-dealer platform. Led end-to-end from discovery research through design, testing, and launch.

**Outcome:** $2.4B in trading volume, $1.3B executed, 37 monthly active users, 45 demos in H1.

### Case Study 2 — From Fragmented to Unified: Automating Rates Derivatives Trading
**JPMorgan Execute · Zero-to-one · Internal sales workflow**

Replacing a 30-year-old legacy application (Sapphire) and migrating the full rates derivatives sales workflow onto Execute. Led discovery research, navigated a major scope pivot, and designed across regulatory infrastructure, structuring complexity, and a modular ticket architecture shared between sales and client experiences.

**Outcome:** In progress.

## How to update

1. Edit files locally or directly on GitHub
2. Upload updated files to the GitHub repository
3. Netlify automatically redeploys — live URL updates in ~30 seconds

## Tech

Plain HTML and CSS — no frameworks, no build step, no dependencies. Every file is self-contained and works by opening in a browser.
